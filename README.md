# Portfolio - Nihanth Bhargav

A modern, responsive portfolio website showcasing projects, skills, and professional experience. Built with React and Vite for optimal performance.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Components** - Smooth animations and transitions for engaging user experience
- **Project Showcase** - Detailed project sections with descriptions and links
- **Skills Display** - Comprehensive skills and technical expertise overview
- **Experience Timeline** - Professional work history and achievements
- **Certifications** - Educational credentials and certifications
- **Contact Section** - Easy way for visitors to reach out
- **SEO Optimized** - Built with best practices for search engine optimization

## Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: CSS3
- **Deployment**: GitHub Pages
- **Fonts**: Google Fonts (Space Mono, Syne)

## Installation

1. Clone the repository:
```bash
git clone https://github.com/NihanthBhargav/Portfolio.git
cd Portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Project Structure

```
Portfolio/
├── src/
│   ├── components/        # React components (Hero, About, Skills, etc.)
│   ├── data/             # Portfolio data and content
│   ├── styles/           # CSS stylesheets
│   ├── App.jsx           # Main app component
│   └── main.jsx          # Entry point
├── public/               # Static assets
├── index.html            # HTML template
├── vite.config.js        # Vite configuration
└── package.json          # Project dependencies
```

## Usage

The portfolio is fully customizable. Update the content by modifying files in the `src/data/` directory to change:
- About section text
- Skills and expertise
- Project details and links
- Work experience
- Certifications
- Contact information

## Deployment

This portfolio is deployed to GitHub Pages. To deploy your own:

1. Update the `homepage` field in `package.json` with your GitHub Pages URL
2. Run the deploy command:
```bash
npm run deploy
```
